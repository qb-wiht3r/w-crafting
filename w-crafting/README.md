# W-Crafting

W-Crafting is a complete and user-friendly crafting system designed for FiveM servers running on **QBCore**.

This script allows players to craft items through an intuitive menu interface, providing a smooth and immersive crafting experience. It supports configurable recipes, crafting requirements, progress animations, and easy customization to fit your server's needs.

### Features

* Fully integrated with **QBCore**
* Uses **qb-menu** for a clean and responsive UI
* Easy-to-configure crafting recipes
* Custom crafting requirements and item costs
* Progress bars and animations
* Optimized and lightweight performance
* Simple installation and configuration

### Dependencies

* qb-core
* qb-menu

Whether you're creating a survival, roleplay, or economy-focused server, W-Crafting provides a flexible and reliable crafting solution for your community.
